# stm32wb09ke-drivers

Bare-metal peripheral drivers for the **STM32WB09KE** (NUCLEO-WB09KE) — register-level drivers and small example projects.

## Summary
This repository contains:
- `drivers/gpio` — GPIO driver implemented
- `drivers/spi` — SPI driver (in progress)
- `drivers/i2c` — I²C driver (planned)
- `Src/001led_toggle` — Toggle LED every one second
- `Src/001led_button` — Toggle LED when user btn is pressed
